# NPC Profiles - 非玩家角色

此目录存放模组 NPC 的数据档案。

## 说明

这些 NPC 档案基于 **Lost Mine of Phandelver（凡戴尔的失落矿坑）** 官方模组创建，包含：

- 主要 NPC（修达、林纳尔、哈比等）
- 反派 NPC（玻璃杖、黑蜘蛛等）
- 地点 NPC（各商店老板、任务给予者）

## 文件格式

每个 NPC 包含：
- 基本信息（姓名、种族、职业）
- 外观描述
- 性格特点
- 知识范围（知道什么/不知道什么）
- 台词示例
- 任务关联

## 使用方式

DM 在扮演这些 NPC 时应参考这些档案，确保：
- 不泄露 NPC 不知道的信息
- 保持性格一致性
- 正确触发任务和剧情

## 模组支持

当前支持：Lost Mine of Phandelver (LMOP)

未来将添加更多官方模组。
